# Gulp Templating - InsanityMeetsHH

## Required
* [Node.js](http://nodejs.org/en/download/ "Node.js")
* [npm](http://www.npmjs.com/get-npm "npm")
* PHP => 5.3

## Install
```bash
$ npm install
```

## Project Commands
See gulpfile.js for further information.
```bash
$ gulp
$ gulp dev
$ gulp prod
$ gulp watch
```
